This Plugin detects mobile browser and display the theme as the setting done from admin. Usefull for switch to Mobile Theme.

== Description ==

Mobilize your WordPress. Detects all the Mobile Browsers and loads the mobile theme as per your settings done in admin.

* Iphone / Ipod
* Ipad
* Android
* Windows Mobile
* Blackberry
* Parm Os
* Opera Mini
* Others Mobile Browsers

You can select a different mobile theme for each mobile browser. You can select different mobile themes for each of the options above.

== Installation ==

1. Upload the plugin wp-mobilizer files to the `/wp-content/plugins/` directory
2. Activate the wp-mobilizer plugin through the 'Plugins' menu in WordPress.
3. Select Themes For Mobile Browsers under WP-Mobilizer > Options

== Frequently Asked Questions ==

= Does it detect all browsers ? =

Yes it detect most of the mobile browsers and redirects it to its mobile theme set from admin settings.
